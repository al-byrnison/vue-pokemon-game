# pokemon-game

## Project Description
Game created based on pokeapi.co and Vue 3, adding unit tests with jest.

## Project Demo

https://vue-choose-pokemon.netlify.app/

## Project setup

```sh
yarn install
```

### Compiles and hot-reloads for development

```sh
yarn serve
```

### Compiles and minifies for production

```sh
yarn build
```

### Run your unit tests

```sh
yarn test:unit
```
